Dynamic Turnstile Environment Setup Completed
Refactored the validation middleware to dynamically pull captcha keys from environment variables (`env.TURNSTILE_SITE_KEY` and `env.TURNSTILE_SECRET_KEY`) instead of hardcoded config databases. Supports automatic ON/OFF bypasses.
env.TURNSTILE_ENABLED = false / true  |  GET /api/auth/branding
Resend Custom Domain Setup Future Task
Currently, email blasts send from `no-reply@7dayscreation.com` using a placeholder credential. To launch, the client needs to set their custom domain DNS records (MX, SPF, DKIM) on Resend.com and configure their real `RESEND_API_KEY` on Cloudflare.
npx wrangler secret put RESEND_API_KEY
